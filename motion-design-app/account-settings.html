To create animations just follow these 5 simple steps:

1.
Visually select an element or class that will trigger the animation
2.
Select a Trigger Type. Choose from click, hover, mouse out, scroll and more...
3.
Select one or more elements that start animating when the trigger fires
4.
Choose an Animation Type like slide in, pulse and many more
5.
Customize the Animation using delays, iterations, offsets, scaling, opacities, colors...
